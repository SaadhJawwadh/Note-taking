# Changelog

## [1.1.0] - 2025-12-15

### Added
- **Tags Support**: Users can now add multiple tags to notes. Tags are displayed on the note cards in the dashboard.
- **Dynamic Theming**: Settings now allow switching between System, Light, and Dark modes.
- **Material You Support**: App icon now supports Android 13+ themed icons (monochrome).
- **Custom Note Colors**: Users can select custom background colors for notes, which reflect in both the editor and the dashboard.
- **Markdown Preview**: Dashboard now displays a snippet of the note content rendered in Markdown.
- **App Icons**: New Light and Dark adaptive app icons.

### Changed
- **UI Refinements**: Updated Toolbar and Note Cards to follow Material 3 design guidelines.
- **Database**: Upgraded database schema to version 3 to support tags.

### Fixed
- **Settings**: Fixed issue where text size setting was not persisting correctly.
